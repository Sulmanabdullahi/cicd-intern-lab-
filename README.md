# cicd-intern-lab

A simple Node.js REST API built during a CI/CD internship lab.

## Routes

| Route | Description |
|---|---|
| `GET /` | Returns a welcome message |
| `GET /health` | Health check |
| `GET /add/:a/:b` | Adds two numbers |

## Setup

```bash
cd cicd-intern-lab
npm install
```

## Run Locally

```bash
node index.js
```

## Run Tests

```bash
npm test
```

## Stack

Node.js · Express · Jest · GitHub Actions · Render.com
